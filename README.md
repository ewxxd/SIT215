# Truck Delivery Search System (SIT215)

## Overview

This project implements a truck delivery navigation system using:

* **Uniform Cost Search (UCS)**
* **A* Search with Manhattan Distance heuristic**

The system finds the shortest path between nodes while respecting **static road constraints**.

---

## How to Run

1. Ensure Python 3 is installed
2. Run the program:

```
222461154.ipynb
```

---

## Requirements

* Python 3.x
* No external libraries required

---

## Notes

* Static constraints are always enforced
* Dynamic constraints are ignored (relaxed problem)
* A* uses Manhattan distance based on node coordinates
